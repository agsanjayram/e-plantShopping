# Paradise Nursery

Paradise Nursery is a React shopping cart application for an online houseplant store. Users can browse plants organized by category, add them to a shopping cart, and manage quantities before checkout.

## Project Name
**e-plantShopping** (Paradise Nursery)

## Features
- Landing page with background image, company description, and a "Get Started" button
- Product listing page with 30 houseplants grouped into 5 categories
- Add to Cart functionality with button disabling once an item is added
- Header/navbar with dynamic cart item count, shown on both Product and Cart pages
- Shopping cart page showing thumbnail, name, unit price, and line total for each item
- Increase / decrease quantity controls and a delete button per cart item
- Total items and total cost displayed prominently
- Continue Shopping and Checkout buttons

## Tech Stack
- React (Vite)
- Redux Toolkit (`@reduxjs/toolkit`, `react-redux`) for cart state management

## Getting Started

Install dependencies:
```
npm install
```

Run the development server:
```
npm run dev
```

Build for production:
```
npm run build
```

## Deployment
This project is deployed using GitHub Pages.
